# Heading Highlighter (Chrome Extension)

A lightweight Chrome extension that highlights all **H1–H6 headings** on any webpage, helping you quickly visualise content structure.

---

## Features

- Highlight H1–H6 tags with distinct colours  
- Toggle each heading level independently  
- Live header count per level  
- Automatically updates on dynamic pages (e.g. SPAs)
- Disable toggles when no headers are present  
- Reset all highlights with one click  

---

## Use Cases

- Frontend development & debugging  
- SEO/content structure analysis  
- Accessibility checks  
- Quickly scanning page hierarchy
